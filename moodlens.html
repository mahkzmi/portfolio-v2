MoodLens AI

Understanding emotions through Natural Language Processing.

MoodLens AI is an NLP-powered sentiment analysis application built using Python and Scikit-learn. It analyzes user-written text and predicts emotional sentiment using a trained machine learning model.

GitHub Back to Portfolio
Project TypeMachine Learning / NLP
StatusCompleted
Development Time1 Week
RoleSolo Developer

The Problem

Many people write journals, notes, or messages without understanding the emotional patterns hidden inside their writing. MoodLens was built to explore how machine learning can classify emotional tone from text in a simple and interactive interface.

Challenges

Selecting an appropriate dataset

Finding a labeled emotion dataset with balanced classes and sufficient variety.

Cleaning noisy text

Removing special characters, stopwords, and handling contractions without losing meaning.

Choosing a lightweight ML model

Balancing accuracy and inference speed for a real-time feel.

Serializing with Pickle

Saving the trained model and vectorizer for portable deployment.

Preparing for deployment

Packaging the app with Streamlit and ensuring a smooth user experience.

The Solution

A complete ML workflow that transforms raw text into emotional insight:

  • Text preprocessing
  • Cleaning
  • Tokenization
  • TF‑IDF Vectorization
  • Logistic Regression
  • Prediction Pipeline
  • Model Serialization
  • Streamlit Interface
